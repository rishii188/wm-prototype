# Results Artifacts

This directory stores generated experiment outputs and analysis artifacts.

## Suggested layout

- `results/*.jsonl` - large experiment outputs (optional; may be ignored by git)
- `results/tables/*.csv` - analysis summary tables generated by `analyze_results.py`
- `results/plots/*.png` - plots generated by `plot_results.py`

## Reproducible analysis workflow

1. Generate summary tables:

```bash
python analyze_results.py --input data/full_run_core_matrix_results.jsonl --out-dir results
```

2. Generate plots from those tables:

```bash
python plot_results.py --results-dir results
```

3. Inspect an individual run/trace:

```bash
python visualize_buffer.py --input data/full_run_core_matrix_results.jsonl --task-id 0
```

For other runs (for example extension outputs), point `--input` to the corresponding JSONL output file.
